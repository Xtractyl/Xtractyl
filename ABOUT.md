## About

Xtractyl is a side project built on weekends by a single developer with a background in clinical medicine, pharmacology research, and evidence evaluation.

The motivation is practical: extracting structured data from unstructured clinical documents is a problem I have encountered repeatedly from different angles. Xtractyl is an attempt to build something that addresses it seriously — with evaluation as a first-class concern, not an afterthought.

The current pipeline covers the full cycle from PDF ingestion to structured output and evaluation. The next step is closing the loop: using validated annotations to finetune a domain-specific model, and serving it locally with output constraints that prevent the model from producing content not present in the input document.

The project is non-commercial and open source. Whether it becomes something more is an open question.
