# Contributing to Xtractyl

Thanks for your interest in contributing to Xtractyl. If you are interested in long term contribution, consider reaching out via email (see at www.xtractyl.com). Details of the project are managed in Jira. Active contributors will be added to the license.

## Prerequisites
- Docker + Docker Compose
- See `.env.example` for required environment variables

## Getting Started
1. Clone the repo
2. Copy `.env.example` to `.env` and fill in the values
3. Run `docker-compose up`

## How to Contribute
- **Bug reports** — open a GitHub Issue with steps to reproduce
- **Feature requests** — open a GitHub Issue describing the use case
- **Pull Requests** — reach out for direct contact first (see above), get Jira acccess

## Areas where contributions are welcome
- Support for complex/long PDFs
- Integration of finetuning

## Code Style
- Python: follow existing structure in `orchestrator/`
- Frontend: template in preparation, js is going to be replaced with ts

## Questions
Open a GitHub Issue or reach out via LinkedIn.
